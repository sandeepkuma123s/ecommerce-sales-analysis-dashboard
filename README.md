# 📊 E-Commerce Sales Analysis Dashboard

## Overview

This project analyzes e-commerce sales data using Python, SQL, and Power BI to uncover valuable business insights. The dashboard helps track revenue, customer behavior, product performance, category-wise sales, and regional trends.

## Project Objectives

* Analyze total sales and revenue
* Identify top-selling products
* Track monthly revenue trends
* Analyze category-wise performance
* Identify best customers
* Perform region-wise sales analysis
* Build an interactive Power BI dashboard

## Tools & Technologies

* Python
* Pandas
* Matplotlib
* SQL (SQLite)
* Power BI
* Git & GitHub
